# Algorand Voting System

## Project Overview
This project demonstrates a simple blockchain based voting system concept using Algorand. 
The goal is to show how blockchain can provide transparency, security and trust in voting systems.

## Features
- Secure voting concept
- Transparent vote counting
- Simple implementation for educational purpose

## Technologies Used
- Python
- Algorand Blockchain (Concept)
- GitHub for version control

## How It Works
1. Users can cast a vote.
2. Votes are stored securely.
3. The system counts the votes and shows the result.

## Future Improvements
- Integration with real Algorand smart contracts
- Web interface for voting
- Wallet based authentication
